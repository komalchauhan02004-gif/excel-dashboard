# Excel Dashboard Project

## Overview
This Excel project is a comprehensive dashboard designed to analyze and visualize sales data. It includes multiple sheets, pivot tables, charts, and interactive elements to provide actionable insights.

The dashboard helps users track key metrics, observe trends, and make data-driven decisions effectively.

---

## Sheets Overview
1. **Sales_Data** – Raw sales data imported for analysis.  
2. **Summary** – Key metrics and KPIs calculated using formulas.  
3. **Pivot_Charts** – Pivot tables and charts for visualizing data.  
4. **Trends** – Trend analysis using line and bar charts.  
5. **Dashboard** – Interactive dashboard summarizing important insights.  
6. **Top 10** – Lists top 10 products, regions, or salespeople based on performance.  
7. **Discount** – Analysis of discounts, offers, and their effect on sales.  
8. **Category Perf** – Performance analysis by product categories.  
9. **Geography** – Sales distribution and analysis by location or region.  
10. **Lookup_Tables** – Supporting tables used for formulas and calculations.  
11. **Analysis** – Detailed calculations, comparisons, and what-if scenarios.

---

## Features
- Interactive Excel dashboard with multiple sheets  
- Dynamic charts and pivot tables  
- Conditional formatting to highlight important data  
- Automated calculations using Excel formulas (SUM, IF, VLOOKUP, INDEX/MATCH)  
- Trend and performance analysis  
- Top 10 performance summary  
- Discount and promotion analysis  
- Category-wise performance analysis  
- Geography-wise sales insights  
- Clean and professional layout for easy interpretation

## Tools Used
- Microsoft Excel  
- Formulas: SUM, IF, VLOOKUP, INDEX/MATCH  
- Pivot Tables & Charts
